The GIL-AUTODOC dictionary


[Generic function]
doc track &key exports paths path slot-way derives-from &allow-other-keys => result


Documents some tracker.


[Method]
doc (track track-package) &key exports paths path slot-way derives-from &allow-other-keys => result


Mention package, but does not mention objects within it.


[Method]
doc (track track-class) &key exports paths path slot-way derives-from &allow-other-keys => result


Document about a class.


[Method]
doc (track track-var) &key exports paths path slot-way derives-from &allow-other-keys => result


Documents a variable.


[Method]
doc (track track-generic) &key exports paths path slot-way derives-from &allow-other-keys => result



[Method]
doc (track track-fun) &key exports paths path slot-way derives-from &allow-other-keys => result



[Method]
doc (track null) &key exports paths path slot-way derives-from &allow-other-keys => result



[Function]
doc-whole-package package &key path-per-obj order state => result



[Function]
gil-autodoc package &key from opts => result


Documents a package, creating a window/file for you.
